## set docker

## Install pipenv (docker で開発するんであれば不要な手順)

```bash
pip install pipenv
```

```bash
pipenv --version
```

- python 3 で動く専用の仮装煥乎油が用意されます
  - pipfile/pipfile.lock が生成される

```bash
pipenv --three
```

- 仮装環境に入ってみる

```bash
pipenv shell
```

- install flask

```bash
pipenv install Flask
```

- install Flask-SQLAlchemy

```bash
pipenv install Flask-SQLAlchemy
```
